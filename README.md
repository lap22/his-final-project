# Hospital Information System (HIS)

## Overview

Hospital Information System (HIS) is a graduation project including:

- Mobile Application (Patient)
- Web Application (Doctor)
- Admin Dashboard
- AI Healthcare Chatbot

## Technologies

### Mobile
- React Native
- Expo
- Redux Toolkit

### Web
- ReactJS
- Material UI

### Backend
- NestJS
- SQL Server
- Prisma ORM
- JWT Authentication

### AI
- OpenAI API

## Main Features

### Patient
- Register/Login
- Book Appointment
- Medical Records
- Appointment History
- AI Symptom Checker

### Doctor
- View Patients
- Create Medical Records
- Diagnosis
- Prescription

### Admin
- Appointment Management
- Doctor Management
- AI Summary
- Dashboard
